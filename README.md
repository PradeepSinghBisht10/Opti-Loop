OptiLoop is a modern, full-stack application designed to solve a critical problem in logistics: finding the shortest path to visit a defined set of mandatory delivery stops before reaching a final destination.

This project serves as a practical demonstration of advanced Data Structures and Algorithms (DSA), utilizing a highly visual and interactive interface to make complex computational solutions accessible to operators. It is built using the MERN stack principles and features Gemini AI integration for enhanced operational intelligence.

TLDR: OptiLoop uses Dijkstra's Algorithm and Brute Force Permutations (TSP) to find the globally optimal delivery circuit, then visualizes the path with an animated truck and provides AI-driven summaries.
